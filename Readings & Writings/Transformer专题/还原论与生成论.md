# 还原论与生成论

还原论主张复杂的现象、系统或规律，最终都可以用更基础、更简单的组成部分来解释。换句话说，要理解整体，先拆解成部分。

而生成论的出发点则偏向于过程。其中的关键词不乏，**涌现、生成性与动态性等**。

在机器翻译的领域中，还原论依靠的是人为设定的规则系统。评分越高，意味着更符合规则。生成论依靠的则是概率学。评分的标准是真实世界的生成偏好，分数越高，意味着更有可能在真实世界中出现。

> 例如，I go this way 和 I this way go
>
> 很明显，第一句评分会更高，不论是在还原论中还是生成论中。它更符合语法的规定，也更多的在真实世界中被人们所使用。

生成论是第一个因果关系完全由数据驱动的范式。因果性不在依赖于人写的规则，而是由“真实数据”决定。数学模型只是把这种偏好显现出来。

如果说还原论体现的是逻辑推理，那么生成论则更注重数据驱动。

这是一种从分析到生成的思维革命。
